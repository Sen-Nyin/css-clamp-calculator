Clamp Calculator

What is this?

CSS clamp allows you to create responsive elements without the need for media queries. It scales your element between a maximum and minimum size as your viewport changes. It also manages the rate at which this scaling happens.

This calculator works out the optimum rate of scaling based on your inputs.

By default, your base text size is 16px, but if you have set your font-size to 62.5%, enter 10.
