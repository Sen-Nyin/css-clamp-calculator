Clamp Calculator

What is this?

CSS clamp allows you to add responsiveness to an element without the need for a media query. The clamp allows your chosen element to scale with your viewport by providing minimum and maximum sizes, as well as the rate at which you want the element to scale.

This calculator takes your min and max sizes and then calculates the optimal rate of scaling.

Remember to add your base text size in pixels. By default this is 16px, but if you have set your font-size to 62.5%, enter 10.
